OPEN24 PROJECT
Overview Reserve Flow GitHub

Open24 is a full booking and operations platform for tennis programs.

The project combines a Nuxt 4 frontend, a Nuxt server BFF, Strapi 5 backend services, and Stripe payments to support court reservations, course enrollments, memberships, credits, and admin/coach operations in one system.

View Reserve Resource Flow Project Org
Client: Nuxt 4 + Nuxt UI v4 + Tailwind 4
BFF: Nuxt server APIs for Stripe + secure backend proxy
Backend: Strapi 5 CMS domain APIs
Payments: Stripe Checkout / PaymentIntents / subscriptions

High-level capabilities

Resource Booking

Users reserve 1-hour court slots with checkout options including cards, credits, and passes.

Course Lifecycle

Supports fixed, ongoing, and on-demand course workflows for customers, coaches, and admins.

Operational Controls

Role-based admin tools for pricing rules, legal docs, session scheduling, and enrollment management.
